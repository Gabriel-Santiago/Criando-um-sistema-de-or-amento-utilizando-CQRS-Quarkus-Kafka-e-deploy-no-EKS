Criando um sistema de orçamento, utilizando CQRS, Quarkus, Kafka e deploy no EKS

Implementamos uma aplicação escrita em Java/Kotlin no serviço Elastic Kubernetes Service da Amazon. A aplicação é um exemplo do padrão CQRS que contempla dois serviços Quarkus que se comunicam através de um barramento assíncrono usando o Kafka. Nós aprendemos a criar os manifestos do Kubernetes para implantação no EKS e quais configurações são necessárias para ter o ambiente rodando em produção.

Tudo isso com o auxilio de Wesley Fuchter.
